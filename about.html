CSS Grid is used on this page to organize images in a consistent and visually balanced layout. Grid properties allow content to be aligned while maintaining flexibility across different screen sizes.

Descendant selectors are applied so that only images within the grid are affected by specific styling rules, ensuring that other layouts such as Flexbox sections remain unaffected.
