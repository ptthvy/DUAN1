Advanced CSS Accessibility and Layout Project

This project demonstrates how Cascading Style Sheets (CSS) can be used to enhance visual design, layout structure, and accessibility across a multi-page website.

  • Home
  • Parks
  • Contact

The primary goal of this assignment is to practice modifying existing CSS rules and adding new ones to improve typography, spacing, color contrast, and overall layout while maintaining semantic HTML structure.

Accessibility considerations are a key focus of this project. Features such as a functional “Skip to Main Content” link and sufficient color contrast help ensure that the site is usable by a wider range of users.
